# api-age
api-age is a PHP Library for get information about your birth date and age

## Installation
```
1️⃣ Upload your host
2️⃣ Get the output as a page request
```
## Example
```
domain.com/age.php?1404/04/04
domain.com/age-2.php?1404/04/04
```
